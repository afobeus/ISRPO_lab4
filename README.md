# План тестирования

### 1. Цели и задачи тестирования
Целью тестирования является обеспечение качества продукта путем проверки работы каждого модуля и выявления потенциальных дефектов на этапе разработки. Основные задачи тестирования:
- Подтвердить, что каждый модуль соответствует заявленным требованиям.
- Обеспечить корректность работы функций на уровне отдельных модулей.
- Гарантировать стабильность и готовность продукта к интеграции с другими компонентами.

### 2. Описание тестируемого продукта
Продукт представляет собой библиотеку для работы с геометрическими объектами. Основная функциональность включает вычисление периметра и площади фигур.

### 3. Область тестирования
В рамках модульного тестирования будут протестированы отдельные компоненты и функции, включая:
- Основные функции библиотеки.
- Логика обработки данных.
- Обработка ошибок и исключений.

### 4. Стратегия тестирования

1. **Модульное тестирование**:
   - **Цель**: Проверка работы каждого модуля или функции в отдельности.
   - **Методы**:
     - Тестирование отдельных функций и методов.

2. **Тестирование граничных значений**:
   - Проверка поведения модулей на границах допустимых значений, таких как минимальные и максимальные значения чисел и т.д.

3. **Проверка на исключения и обработку ошибок**:
   - Тестирование корректной обработки ошибок и ситуаций, когда выбрасываются исключения при вводе недопустимых данных.

4. **Рефакторинг с последующей проверкой**:
   - После изменений проводится повторное тестирование, чтобы убедиться, что рефакторинг не нарушил работу модулей.

5. **Покрытие кода тестами**:
   - Цель — достичь высокого покрытия тестами для важных частей кода.

### 5. Критерии приемки
Продукт будет считаться готовым к интеграционному тестированию, если:
- Все модули успешно проходят модульное тестирование без ошибок.
- Обработка ошибок и граничные значения протестированы и работают корректно.

### 6. Ожидаемые результаты
Результаты тестирования будут включать:
- Отчеты о тестировании с выявленными дефектами и их статусами.
- Степень покрытия тестами.
